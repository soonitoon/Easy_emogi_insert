# Easy_emogi_insert
This program helps you to insert emoji easy way.

## key
- F1 : ❤
- F2 : ☺
- F3 : 😆
- F4 : 🥰
- F5 : 😚
- F6 : 😫
- F7 : 😢
- F8 : 😶
- F9 : 🔥
- F11 : ⭐
- F12 : **quit**

## Notice
you need python mudules
- pyautogui `$ python -m pip install pyautogui`
- pyperclip `$ python -m pip install pyperclip`
- keyboard `$ python -m pip install keyboard`
